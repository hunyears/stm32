## 📝 学习建议

1. **先理解原理，再写代码** — 每次操作前先看参考手册对应章节
2. **画图辅助理解** — 用方框图画出信号流向
3. **动手验证** — 每学一个知识点，写代码烧录验证
4. **记录问题** — 遇到 bug 记录原因和解决方法，比笔记更有价值
5. **定期回顾** — 每周回顾之前的内容，温故知新
