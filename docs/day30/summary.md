## 二、进阶方向

### 2.1 RTOS（实时操作系统）

```
FreeRTOS：
    - 任务调度
    - 信号量/互斥量
    - 队列通信
    - 软件定时器

学习资源：
    - 《FreeRTOS 内核实现与应用开发实战指南》
    - 官方文档：freertos.org
```

### 2.2 USB 开发

```
USB 外设：
    - HID 设备（键盘、鼠标）
    - CDC 类（虚拟串口）
    - MSC 类（大容量存储）

学习要点：
    - USB 协议基础
    - 描述符配置
    - 端点管理
```

### 2.3 以太网

```
网络协议栈：
    - LwIP
    - TCP/UDP 通信
    - HTTP/MQTT

应用场景：
    - 物联网设备
    - 远程监控
    - 数据采集
```

### 2.4 嵌入式 Linux

```
进阶路径：
    - Bootloader 开发
    - 驱动开发
    - 应用程序开发

推荐资源：
    - 《嵌入式 Linux 应用程序开发标准教程》
```
