## 一、项目概述

### 1.1 项目目标

设计一个智能温度控制系统，实现：
- 环境温度监测
- 温度超限报警
- 自动控制加热/制冷设备
- 数据显示和远程监控

## 二、需求分析

### 2.1 功能需求

```
1. 温度监测
   - 实时显示温度
   - 精度 ±0.5°C
   - 采样周期 1 秒

2. 温度控制
   - 设定目标温度
   - 自动加热/制冷
   - 温度区间报警

3. 用户交互
   - OLED 显示界面
   - 按键参数设置
   - 状态指示

4. 数据通信
   - 串口数据上报
   - 远程参数设置
```

### 2.2 技术选型

```
主控芯片：STM32F103C8T6
温度传感器：AHT10（I2C）
显示模块：0.96寸 OLED（I2C）
控制元件：继电器模块
报警元件：有源蜂鸣器
通信接口：USART1
```

## 三、软件架构

### 3.1 主循环结构

```c
int main(void)
{
    // 系统初始化
    System_Init();

    while (1)
```

### 3.2 状态机设计

```c
typedef enum {
    STATE_IDLE,      // 空闲状态
    STATE_HEATING,   // 加热状态
    STATE_COOLING,   // 制冷状态
    STATE_ALARM      // 报警状态
} System_State;

void StateMachine_Process(void)
{
    float temp = Get_Temperature();

    switch (current_state)
    {
        case STATE_IDLE:
            if (temp < target_temp - 2.0f)
                current_state = STATE_HEATING;
            else if (temp > target_temp + 2.0f)
                current_state = STATE_COOLING;
            break;

        case STATE_HEATING:
            Relay_On(RELAY_HEAT);
            if (temp >= target_temp)
            {
                Relay_Off(RELAY_HEAT);
                current_state = STATE_IDLE;
            }
            break;

        case STATE_COOLING:
            Relay_On(RELAY_COOL);
            if (temp <= target_temp)
            {
                Relay_Off(RELAY_COOL);
                current_state = STATE_IDLE;
            }
            break;

        case STATE_ALARM:
            Buzzer_On();
            break;
    }
}
```
